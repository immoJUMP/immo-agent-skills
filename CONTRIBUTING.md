# Mitmachen

Beitraege zu den Immo Agent Skills sind herzlich willkommen! Dieses Projekt lebt davon, dass Praktiker ihr Wissen einbringen.

## Was wir suchen

- **Neue Skills** fuer weitere Immobilien-Prozesse
- **Verbesserungen** bestehender Skills (genauere Formeln, bessere Prompts, fehlende Schritte)
- **Regionale Benchmarks** (Mietspiegel-Daten, lokale Sanierungskosten, regionale Besonderheiten)
- **Fehlerkorrekturen** (veraltete Gesetze, falsche Berechnungen, Tippfehler)
- **Praxiserfahrungen** (Was hat bei euch funktioniert? Was nicht?)

## Wie beitragen

1. Repository forken
2. Neuen Branch erstellen (`feature/neuer-skill-name`)
3. Aenderungen vornehmen
4. Pull Request erstellen mit kurzer Beschreibung

## Richtlinien

### Skill-Dateien

- Alle Skills folgen der Standardstruktur (siehe bestehende Skills als Vorlage)
- Prosa in Deutsch, JSON-Keys in Englisch
- Konkrete Zahlen und Schwellenwerte statt vager Beschreibungen
- Gesetzesreferenzen mit Paragraphen angeben (z.B. §558 BGB)
- Keine personenbezogenen Daten in Beispielen

### Knowledge-Dateien

- Quellen angeben wo moeglich
- Datum der letzten Aktualisierung vermerken
- Gesetzesaenderungen zeitnah einpflegen

### Qualitaet

- Jeder Skill muss mit mindestens einem realen Beispiel getestet sein
- Berechnungen muessen mathematisch korrekt sein
- Rechtliche Referenzen muessen dem aktuellen Stand entsprechen

## Verhaltenskodex

- Respektvoller Umgang miteinander
- Konstruktive Kritik statt Ablehnung
- Praxis vor Theorie
- Keine Werbung fuer einzelne Produkte oder Dienstleistungen in den Skills
