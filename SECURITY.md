# Sicherheitsrichtlinie

## Ueber dieses Projekt

Dieses Repository enthaelt ausschliesslich Markdown-Dateien: Skill-Beschreibungen und Wissensdatenbanken fuer die Immobilienanalyse. Es gibt keinen ausfuehrbaren Code, keine API-Endpunkte, keine Datenbanken und keine Benutzerauthentifizierung.

## Sicherheitsproblem melden

Falls Sie ein Sicherheitsproblem in diesem Projekt entdecken -- z.B. versehentlich eingecheckte sensible Daten, ein Link der auf eine schaedliche Seite weiterleitet, oder personenbezogene Daten in Beispieldateien -- melden Sie dies bitte per E-Mail an:

**security@immojump.de**

Sie erhalten innerhalb von 48 Stunden eine Antwort. Wird das Problem bestaetigt, wird ein Fix veroeffentlicht und der Melder namentlich erwaehnt (sofern keine Anonymitaet gewuenscht wird).

## Angriffsflaeche

Da dieses Projekt ausschliesslich Markdown-Dokumentationsdateien enthaelt, beschraenkt sich die Angriffsflaeche auf:

- Schaedliche Links in Skill-Dateien
- Versehentlich in Beispiel-Vorlagen enthaltene sensible Daten
- Personenbezogene Daten in Beispiel-Eingaben
- Phishing- oder Identitaetsmissbrauch

Falls Sie eines der oben genannten Probleme finden, melden Sie es bitte.

## Datenschutzhinweis

Die Beispiel-Eingaben in `templates/beispiel-inputs/` enthalten ausschliesslich fiktive Daten. Es werden keine echten Mieterdaten, Adressen oder personenbezogenen Informationen in diesem Repository gespeichert.
