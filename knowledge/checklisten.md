# Checklisten - Praxisleitfaden fuer deutsche Wohnimmobilien

## 1. Ankauf-Checkliste: Welche Unterlagen anfordern?

### Vom Verkaeufer / Makler anfordern

**Grunddaten und Eigentum:**
- [ ] Aktueller Grundbuchauszug (nicht aelter als 3 Monate)
- [ ] Teilungserklaerung (bei WEG) inkl. aller Nachtraege und Aenderungen
- [ ] Gemeinschaftsordnung
- [ ] Aufteilungsplan mit Sondereigentumseinheiten
- [ ] Abgeschlossenheitsbescheinigung
- [ ] Lageplan / Flurkarte

**Bau und Zustand:**
- [ ] Grundrisse (alle Geschosse, Keller, Dach)
- [ ] Baubeschreibung / Baujahr-Nachweis
- [ ] Energieausweis (Verbrauchs- oder Bedarfsausweis)
- [ ] Dokumentation durchgefuehrter Sanierungen und Modernisierungen
- [ ] Brandschutzgutachten (bei MFH)
- [ ] Statische Unterlagen (bei aelteren Gebaeuden oder Umbauten)
- [ ] Altlastenbescheinigung / Baugrundgutachten (falls vorhanden)
- [ ] Denkmalschutzbescheid (falls zutreffend)

**Mietverhaeltnisse:**
- [ ] Mieterliste mit Miethoehe, Mietbeginn, Wohnflaeche, Zimmeranzahl
- [ ] Kopien aller Mietvertraege inkl. Nachtraege
- [ ] Kautionskonten / Kautionsuebersicht
- [ ] Mietrueckstandsuebersicht (mind. letzte 12 Monate)
- [ ] Betriebskostenabrechnungen (letzte 2-3 Jahre)
- [ ] Mietspiegelvergleich / Mietsteigerungspotenzial

**WEG-Unterlagen (bei Eigentumswohnung):**
- [ ] Protokolle der letzten 3-5 Eigentuemversammlungen
- [ ] Wirtschaftsplan (aktuell und Vorjahr)
- [ ] Jahresabrechnung (letzte 2-3 Jahre)
- [ ] Stand der Instandhaltungsruecklage
- [ ] Beschlusssammlung
- [ ] Angaben zu beschlossenen oder geplanten Sonderumlagen
- [ ] Hausordnung
- [ ] Verwaltervertrag (Name, Laufzeit, Kosten)

**Finanzen und Steuern:**
- [ ] Grundsteuerbescheid
- [ ] Uebersicht laufender Kosten (Hausgeld, Versicherungen, etc.)
- [ ] Bestehende Versicherungsvertraege (Gebaeude, Haftpflicht)
- [ ] Wartungsvertraege (Heizung, Aufzug, Garten, etc.)

## 2. Besichtigungs-Checkliste: Was vor Ort pruefen?

### Aussenbereich (10 Minuten)

**Umgebung:**
- [ ] Laerm (Strasse, Gewerbe, Bahn, Flughafen)?
- [ ] Parkmoeglichkeiten
- [ ] OEPNV-Anbindung (Entfernung Haltestelle)
- [ ] Nahversorgung (Supermarkt, Aerzte, Apotheke)
- [ ] Schulen, Kindergaerten in der Naehe
- [ ] Gesamteindruck der Nachbarschaft
- [ ] Zustand der Nachbarhaeuser

**Gebaeude aussen:**
- [ ] Fassadenzustand (Risse, Abplatzungen, Feuchtigkeit, Algen)
- [ ] Daemmung vorhanden? Zustand?
- [ ] Dachzustand (Ziegel, Rinnen, Fallrohre) - von der Strasse beurteilbar?
- [ ] Fenster (Alter, Material, Verglasung, Zustand der Dichtungen)
- [ ] Balkon/Terrasse (Bodenbelag, Gelaender, Abdichtung)
- [ ] Eingangstuer und Schliesssystem
- [ ] Briefkastenanlage
- [ ] Muelltonnenstellplatz
- [ ] Fahrradstellplaetze
- [ ] Garten/Aussenanlagen (Pflege, Zustand)

### Gemeinschaftsbereiche (10 Minuten)

- [ ] Treppenhaus (Zustand, Sauberkeit, Beleuchtung)
- [ ] Kellerflure (Feuchtigkeit, Geruch, Belueftung)
- [ ] Kellerparzelle der Wohnung (Groesse, Zustand)
- [ ] Dachboden (begehbar? Zustand? Ausbaubar?)
- [ ] Waschkueche/Trockenraum
- [ ] Aufzug (Baujahr, letzte Pruefung, Zustand)
- [ ] Tiefgarage / Stellplaetze
- [ ] Heizungsraum (Heizungstyp, Alter, Wartungszustand)
- [ ] Zaehler (Strom, Gas, Wasser - Einzelzaehler vorhanden?)
- [ ] Brandschutzeinrichtungen (Feuerloescher, Fluchtweg-Beschilderung)

### Wohnung innen (20 Minuten)

**Raeume allgemein:**
- [ ] Raumgroesse und Raumaufteilung praxistauglich?
- [ ] Deckenhoehe
- [ ] Bodenbelaege (Zustand, Art, Alter)
- [ ] Waende und Decken (Risse, Feuchtigkeit, Schimmel)
- [ ] Fenster pruefen: Oeffnen, Schliessen, Dichtungen, Schwitzwasser
- [ ] Tueren: Gangbarkeit, Schliessmechanismus
- [ ] Steckdosen und Lichtschalter (Anzahl, Zustand, Sicherungskasten)
- [ ] Heizkoerper (Typ, Alter, Funktion, Thermostate)
- [ ] Belueftung (Fenster in jedem Raum? Innenliegendes Bad?)

**Kueche:**
- [ ] Zustand der Einbaukueche (falls vorhanden)
- [ ] Wasseranschluesse (Warm/Kalt)
- [ ] Abwasseranschluss
- [ ] Strom (ausreichend Anschluesse? Starkstrom fuer Herd?)
- [ ] Dunstabzug (Abluft oder Umluft?)

**Bad/WC:**
- [ ] Zustand Sanitaerobjekte (Badewanne, Dusche, WC, Waschbecken)
- [ ] Fliesen (Zustand, Fugen)
- [ ] Belueftung (Fenster oder Luefter?)
- [ ] Wasserdruck pruefen (Hahn aufdrehen)
- [ ] Warmwasserversorgung (Zentral oder Durchlauferhitzer/Boiler?)
- [ ] Waschmaschinenanschluss vorhanden?
- [ ] Silikon-Fugen (Schimmel, Zustand)

**Spezifische Pruefpunkte:**
- [ ] Geruch (Schimmel, Feuchtigkeit, Rauch, Abwasser)
- [ ] Lichtverhaeltnisse (Tageslicht, Ausrichtung)
- [ ] Laermpegel innen (Trittschall, Nachbarn, Strasse)
- [ ] Internetverfuegbarkeit (Glasfaser? DSL-Geschwindigkeit?)
- [ ] Rauchwarnmelder installiert?

### Fotos und Dokumentation

- [ ] Alle Raeume fotografieren
- [ ] Maengel und Schaeden detailliert fotografieren
- [ ] Zaehlerstaende notieren
- [ ] Heizungstyp und Baujahr notieren
- [ ] Energieausweis-Daten notieren
- [ ] Gesamteindruck und Bauchgefuehl notieren

## 3. Uebergabe-Checkliste: Ein-/Auszug

### Auszug des bisherigen Mieters

**Vorbereitung:**
- [ ] Kuendigungsschreiben pruefen (fristgerecht? formgueltig?)
- [ ] Termin fuer Wohnungsuebergabe vereinbaren
- [ ] Vorabbegehung 4-6 Wochen vor Auszug (Zustand pruefen, Maengel besprechen)
- [ ] Mieter an Schoenheitsreparatur-Pflicht erinnern (falls wirksam vereinbart)
- [ ] Kautionsabrechnung vorbereiten

**Uebergabe:**
- [ ] Uebergabeprotokoll erstellen (Datum, Uhrzeit, Anwesende)
- [ ] Alle Zaehlerstaende ablesen und dokumentieren (Strom, Gas, Wasser, Heizung)
- [ ] Zustand jedes Raumes dokumentieren (schriftlich + Fotos)
- [ ] Alle Schluessel zuruecknehmen (Haustuer, Wohnung, Keller, Briefkasten, Garage)
- [ ] Anzahl der Schluessel mit Mietvertrag abgleichen
- [ ] Schaeden und Maengel dokumentieren (mit Fotos)
- [ ] Unterschrift beider Parteien auf Uebergabeprotokoll
- [ ] Nachsendeauftrag des Mieters pruefen
- [ ] Versorger ueber Mieterwechsel informieren
- [ ] Kautionsabrechnung erstellen (fristgerecht, typisch 3-6 Monate)

### Einzug des neuen Mieters

**Vorbereitung:**
- [ ] Mietvertrag unterschrieben?
- [ ] Kaution erhalten? (Erste Rate bei Einzug, weitere in 2 Monatsraten)
- [ ] Kautionskonto eroeffnet?
- [ ] Wohnung renoviert/gereinigt?
- [ ] Alle Maengel aus Vorabnahme behoben?
- [ ] Schluessel vorbereitet (Beschriftung, Anzahl)
- [ ] Hausverwaltung ueber neuen Mieter informiert

**Uebergabe:**
- [ ] Uebergabeprotokoll erstellen
- [ ] Alle Zaehlerstaende ablesen und dokumentieren
- [ ] Zustand jedes Raumes dokumentieren
- [ ] Alle Schluessel uebergeben und dokumentieren
- [ ] Vorhandene Maengel im Protokoll festhalten
- [ ] Hausordnung uebergeben
- [ ] Wichtige Informationen mitteilen (Muelltermine, Heizungsanlage, Ansprechpartner)
- [ ] Ummeldebescheinigung des Einwohnermeldeamtes einfordern
- [ ] Unterschrift beider Parteien

## 4. Vermietungs-Checkliste: Von Inserat bis Einzug

### Phase 1: Vorbereitung

- [ ] Marktmiete recherchieren (Mietspiegel, Immobilienportale, Vergleichsangebote)
- [ ] Mietpreisbremse pruefen (Gebiet mit angespanntem Wohnungsmarkt?)
- [ ] Zulaessige Miete berechnen (Vergleichsmiete + ggf. 10% Aufschlag)
- [ ] Wohnung fuer Besichtigung vorbereiten (sauber, aufgeraeumt, gut beleuchtet)
- [ ] Professionelle Fotos erstellen (Tageslicht, alle Raeume, Aussenansicht)
- [ ] Grundriss erstellen oder aktualisieren
- [ ] Energieausweis bereithalten (Pflicht!)
- [ ] Exposee erstellen mit allen relevanten Daten

### Phase 2: Vermarktung

- [ ] Inserat auf Immobilienportalen schalten (ImmoScout24, Immowelt, etc.)
- [ ] Inserattext: Lage, Groesse, Zimmer, Ausstattung, Miete, Nebenkosten, Kaution
- [ ] Energieausweisdaten im Inserat angeben (Pflicht! Bussgeld bis 15.000 EUR)
- [ ] Besichtigungstermine planen (Einzel- oder Sammelbesichtigungen)
- [ ] Alternativ: Makler beauftragen (Kosten traegt Vermieter bei Neuvermietung)

### Phase 3: Mieterauswahl

- [ ] Selbstauskunft der Interessenten einholen
- [ ] SCHUFA-Auskunft anfordern (Bonitaetspruefung)
- [ ] Einkommensnachweis (letzte 3 Gehaltsnachweise oder Steuerbescheid)
- [ ] Mietschuldenfreiheitsbescheinigung des Vorvermieters
- [ ] Personalausweis / Aufenthaltstitel pruefen
- [ ] Arbeitgeberbescheinigung (bei Angestellten)
- [ ] Bei Selbstaendigen: BWA oder Steuerbescheid
- [ ] Anzahl der einziehenden Personen klaeren
- [ ] Haustiere klaeren
- [ ] Datenschutz beachten (nur erforderliche Daten erheben)

### Phase 4: Vertragsschluss

- [ ] Mietvertrag erstellen (aktuelles Muster, rechtssicher)
- [ ] Miethoehe, Nebenkosten, Kaution festlegen
- [ ] Nebenkostenvorauszahlung oder -pauschale?
- [ ] Staffelmiete oder Indexmiete vereinbaren? (optional)
- [ ] Schoenheitsreparaturklausel pruefen (aktuelle Rechtsprechung!)
- [ ] Haustierhaltung regeln
- [ ] Untervermietungsklausel
- [ ] Kleinreparaturklausel (max. 75-120 EUR/Einzelfall, max. 200-300 EUR/Jahr, max. 8% der Jahresmiete)
- [ ] Mietvertrag von beiden Parteien unterschreiben lassen
- [ ] Kautionsvereinbarung treffen

### Phase 5: Einzug

- [ ] Wohnungsuebergabe (siehe Uebergabe-Checkliste)
- [ ] Hausverwaltung informieren
- [ ] Versorger informieren (Strom, Gas, Wasser)
- [ ] Versicherung aktualisieren (Vermieterwechsel/neuer Mieter)
- [ ] Mieterakte anlegen (Vertrag, Protokoll, Korrespondenz)

## 5. Verwaltungs-Checkliste: Monatliche und jaehrliche Pflichten

### Monatliche Aufgaben

- [ ] Mieteingang pruefen (bis zum 3. Werktag)
- [ ] Mietrueckstaende identifizieren und Mahnwesen einleiten
- [ ] Handwerkeranfragen und Reparaturauftraege bearbeiten
- [ ] Mieteranfragen beantworten
- [ ] Nebenkostenvorauszahlungen kontrollieren
- [ ] Leerstandsmanagement (Vermarktung freier Einheiten)
- [ ] Kautionskonten verwalten
- [ ] Buchfuehrung (Einnahmen/Ausgaben)

### Quartalsweise Aufgaben

- [ ] Objektbegehung (Zustand Gemeinschaftsflaechen, Aussenanlagen)
- [ ] Wartungsvertraege pruefen (Heizung, Aufzug, Rauchmelder)
- [ ] Ruecklagenstatus pruefen
- [ ] Versicherungsdeckung pruefen
- [ ] Nebenkostenbudget pruefen (Soll/Ist-Vergleich)

### Jaehrliche Aufgaben

**1. Quartal (Januar-Maerz):**
- [ ] Betriebskostenabrechnung Vorjahr erstellen (Frist: 12 Monate nach Abrechnungszeitraum)
- [ ] Nebenkostenvorauszahlungen anpassen (falls erforderlich)
- [ ] Grundsteuerbescheid pruefen
- [ ] Versicherungen pruefen und ggf. anpassen

**2. Quartal (April-Juni):**
- [ ] Mietsteigerungspotenzial pruefen (Mietspiegel, Vergleichsmieten)
- [ ] Mieterhoehungsverlangen vorbereiten (falls zulaessig)
- [ ] Eigentuemerversammlung vorbereiten/teilnehmen (WEG)
- [ ] Fruehjahrsbegehung (Winter-/Frostschaeden?)

**3. Quartal (Juli-September):**
- [ ] Instandhaltungsplanung fuer Folgejahr
- [ ] Sanierungsmassnahmen planen und Angebote einholen
- [ ] Mietvertraege pruefen (Indexanpassung, Staffelmiete)
- [ ] Rauchmelder pruefen (laenderspezifisch, meist jaehrlich)

**4. Quartal (Oktober-Dezember):**
- [ ] Winterdienst organisieren (Streupflicht!)
- [ ] Heizungsanlage warten lassen (vor Beginn Heizperiode)
- [ ] Steuerliche Unterlagen vorbereiten (fuer Steuererklaerung)
- [ ] Wirtschaftsplan Folgejahr pruefen (WEG)
- [ ] Budgetplanung Folgejahr

### Anlass-bezogene Aufgaben

- [ ] Mieterwechsel: Kuendigung, Abnahme, Renovierung, Neuvermietung, Uebergabe
- [ ] Wasserschaden/Notfall: Sofortmassnahmen, Versicherung, Handwerker
- [ ] Mieterhoehung: Mietspiegel pruefen, Schreiben aufsetzen, Fristen beachten
- [ ] Modernisierung: Ankuendigung (3 Monate vorher), Durchfuehrung, Umlage
- [ ] Rechtsstreit: Dokumentation, Anwalt, Fristen

## 6. Due-Diligence-Checkliste: Vollstaendige Dokumentenpruefung

### A. Rechtliche Due Diligence

**Grundbuch:**
- [ ] Abteilung I: Eigentuemer korrekt? Erbengemeinschaft? GbR?
- [ ] Abteilung II: Lasten und Beschraenkungen (Wegerechte, Leitungsrechte, Niessbbrauch, Wohnrecht, Vormerkungen, Baulasten)
- [ ] Abteilung III: Grundschulden/Hypotheken (Werden bei Verkauf geloescht?)
- [ ] Altlasten im Grundbuch oder Baulastenverzeichnis?

**Baurecht:**
- [ ] Baugenehmigungen vorhanden? Fuer alle Um-/Anbauten?
- [ ] Nutzungsaenderungen genehmigt?
- [ ] Bebauungsplan pruefen (Nutzungsart, GRZ, GFZ, Geschossigkeit)
- [ ] Denkmalschutz? Gesamtanlage oder Einzeldenkmal?
- [ ] Baulasten (Baulastenverzeichnis bei der Bauaufsicht)

**Mietrecht:**
- [ ] Alle Mietvertraege auf Wirksamkeit pruefen
- [ ] Schoenheitsreparaturklauseln wirksam?
- [ ] Miethoehe zulaessig (Mietpreisbremse)?
- [ ] Kappungsgrenze beachtet bei frueheren Erhoehungen?
- [ ] Laufende Rechtsstreitigkeiten mit Mietern?
- [ ] Kuendigungen ausgesprochen?
- [ ] Mietminderungen laufend?

**WEG-Recht:**
- [ ] Teilungserklaerung auf Sondernutzungsrechte pruefen
- [ ] Oeffnungsklauseln in der Gemeinschaftsordnung?
- [ ] Beschlusssammlung auf relevante Beschluesse pruefen
- [ ] Angefochtene Beschluesse?
- [ ] Verwaltervertrag: Laufzeit, Kuendigungsmoeglichkeit

### B. Technische Due Diligence

**Gebaeude:**
- [ ] Baujahr und Bausubstanz (Massiv, Fertigbau, Fachwerk?)
- [ ] Letzte Sanierungsmassnahmen (wann? was? Qualitaet?)
- [ ] Dachzustand und Restlebensdauer
- [ ] Fassade und Daemmzustand
- [ ] Kellerabdichtung und Feuchtigkeitssituation
- [ ] Fenster (Alter, Verglasung, Zustand)
- [ ] Schadstoffbelastung (Asbest, KMF, PCB, PAK, Holzschutzmittel)

**Haustechnik:**
- [ ] Heizungsanlage (Typ, Alter, Zustand, GEG-Konformitaet)
- [ ] Elektroinstallation (Alter, Zustand, FI-Schutzschalter)
- [ ] Wasser- und Abwasserleitungen (Material, Alter, Zustand)
- [ ] Aufzug (Baujahr, letzte TUEV-Pruefung, Zustand)
- [ ] Lueftungsanlage (falls vorhanden)

**Energetik:**
- [ ] Energieausweis (Verbrauch oder Bedarf)
- [ ] Energieeffizienzklasse
- [ ] GEG-Anforderungen: Welche Pflichten bestehen?
- [ ] Foerderfaehigkeit fuer energetische Sanierung pruefen
- [ ] CO2-Kosten-Aufteilung (seit 01.01.2023, nach CO2KostAufG)

### C. Wirtschaftliche Due Diligence

**Mieteinnahmen:**
- [ ] IST-Miete vs. SOLL-Miete vs. ortsuebliche Vergleichsmiete
- [ ] Mietsteigerungspotenzial berechnen (Kappungsgrenze beachten!)
- [ ] Leerstandsrisiko bewerten
- [ ] Mietausfaelle der letzten 3 Jahre pruefen
- [ ] Bonitat der Mieter bewerten

**Kosten:**
- [ ] Hausgeld-Entwicklung der letzten 3-5 Jahre
- [ ] Sonderumlagen der Vergangenheit und geplante
- [ ] Instandhaltungsruecklage: Hoehe und Historie
- [ ] Nicht umlagefaehige Kosten genau beziffern
- [ ] Verwaltungskosten marktgerecht?
- [ ] Versicherungskosten angemessen?

**Kalkulation:**
- [ ] Vollstaendige Kaufpreiskalkulation inkl. Nebenkosten
- [ ] Brutto- und Nettomietrendite berechnen
- [ ] Cashflow-Analyse (monatlich und jaehrlich)
- [ ] Finanzierbarkeit pruefen (Eigenkapitalbedarf, Beleihungsauslauf)
- [ ] Steuerliche Auswirkungen berechnen (AfA, Zinsabzug, Cashflow nach Steuern)
- [ ] EK-Rendite und EK-Rueckfluss berechnen
- [ ] Sanierungskosten einpreisen
- [ ] Best/Worst/Base-Case-Szenarien rechnen
- [ ] DSCR berechnen
- [ ] Sensitivitaetsanalyse (Zinsaenderung, Mietausfall, Leerstand)

### D. Standort Due Diligence

- [ ] Bevoelkerungsentwicklung (Prognose 10-20 Jahre)
- [ ] Wirtschaftsstruktur und Arbeitsmarkt
- [ ] Mietpreisentwicklung der letzten 5 Jahre
- [ ] Kaufpreisentwicklung der letzten 5 Jahre
- [ ] Leerstandsentwicklung
- [ ] Infrastrukturprojekte geplant (OEPNV, Strassen, Uni, Gewerbe)?
- [ ] Sozialstruktur und Migrationsströme
- [ ] Konkurrenzangebot (Neubauprojekte in der Umgebung?)
- [ ] Mikrolage: Laermbelastung, Nachbarschaft, Image des Viertels
- [ ] Makrolage: Anbindung an ueberregionale Infrastruktur

## 7. Schnellcheck fuer Erstbewertung (5-Minuten-Pruefung)

Vor detaillierter Analyse mit diesen 10 Fragen grob filtern:

| Nr. | Frage | Ergebnis |
|---|---|---|
| 1 | Kaufpreisfaktor < 25? | Ja/Nein |
| 2 | Bruttomietrendite > 4%? | Ja/Nein |
| 3 | Lage: Bevoelkerungswachstum positiv? | Ja/Nein |
| 4 | Baujahr/Zustand: Keine sofortige Kernsanierung noetig? | Ja/Nein |
| 5 | Mietverhaeltnisse: Kein Grossleerstand (> 20%)? | Ja/Nein |
| 6 | Energieausweis: Nicht Klasse G oder H? | Ja/Nein |
| 7 | Grundbuch: Keine offensichtlichen Belastungen? | Ja/Nein |
| 8 | WEG: Keine exorbitante Sonderumlage angekuendigt? | Ja/Nein |
| 9 | Hausgeld: < 4 EUR/qm/Monat? | Ja/Nein |
| 10 | Finanzierbar: Eigenkapital fuer mind. Nebenkosten vorhanden? | Ja/Nein |

**Bewertung:**
- 8-10x Ja: Detailanalyse lohnt sich
- 5-7x Ja: Genauer hinsehen, Risiken identifizieren
- < 5x Ja: Wahrscheinlich kein attraktives Investment
